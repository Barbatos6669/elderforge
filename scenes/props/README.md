# Prop Scenes

Reusable world prop prefabs live here. These scenes should be safe to drop under
`World/LevelContent/Props`, `Buildings`, or another level content folder.

- `ruin_arch/LowPolyRuinArch.tscn`: low-poly magical ruin arch with source art
  in `assets/models/props/ruin_arch/`.
- `fantasy_props_megakit/`: wrapper prefabs for the imported Fantasy Props
  MegaKit models.
- `free_3d_modular_game_assets_for_prototyping/`: wrapper prefabs for the CC0
  modular prototyping pack.
