# Universal Base Character Package

Imported from the Desktop `godot packages` folder:

- `Universal Base Characters[Standard].zip`
- `Modular Character Outfits - Fantasy[Standard].zip`

This folder preserves the original package layout so we can inspect base bodies,
hair, complete outfits, and modular outfit pieces without changing the current
player prefab.

Godot-friendly meshes are usually under folders named `glTF (Godot...)`.
Unity/FBX exports are kept as source references for Blender or future conversion.
