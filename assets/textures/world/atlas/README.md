# World Texture Atlas

This folder owns the shared atlas for outdoor terrain and city/world props.

Tile IDs:

- `grass_meadow`
- `worn_grass`
- `packed_dirt`
- `dirt_path`
- `cut_stone`
- `cobblestone`
- `ruin_stone`
- `dark_cracked_stone`
- `timber_planks`
- `tree_bark`
- `moss`
- `vines`
- `sand`
- `cliff_rock`
- `blue_rune_stone`
- `gold_trim`

Use `world_texture_atlas.json` as the source of truth for UV rectangles. The
preview PNG has labels and should not be used on runtime meshes.
