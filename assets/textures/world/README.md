# World Textures

World textures are shared by terrain, city pieces, ruins, roads, and other
environment art. Prefer atlases here when a group of modular meshes can share a
single material.

Current atlas:

- `atlas/world_texture_atlas.png`: runtime texture for world meshes.
- `atlas/world_texture_atlas_preview.png`: labeled artist reference only.
- `atlas/world_texture_atlas.json`: tile names, pixel rectangles, and UVs.
- `res://assets/materials/world/world_texture_atlas_material.tres`: starter
  Godot material that points at the atlas.

The atlas is generated by:

```powershell
python tools/art/create_world_texture_atlas.py
```

Atlas layout uses 256 px cells with an 8 px gutter. Use the `uv_rect` values in
`world_texture_atlas.json` when assigning UVs so mipmaps do not bleed between
neighboring tiles.

The current texture pass is a programmer-art foundation. It is meant to be
stable enough for UVs and modular level building, then repainted or replaced as
the world art direction matures.

Water:

- `water/`: generated tileable normal and foam maps used by
  `res://assets/materials/world/stylized_water.tres`.
