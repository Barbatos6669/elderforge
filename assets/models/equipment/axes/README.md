# Legacy Axe Model Assets

This folder holds the original single T1 axe placeholder.

New axe work should happen under:

```text
assets/equipment/tools/axes/t1/
assets/equipment/tools/axes/t2/
...
assets/equipment/tools/axes/t8/
```

Those production tier folders contain separate `.blend` sources and `.glb`
exports for each tier. The old files here are kept only so we do not destroy
previous local work while the project moves to the tiered equipment layout.
