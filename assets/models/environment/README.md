# Environment Models

Environment models are reusable world-building assets such as trees, rocks,
foliage, ruins, roads, and city decoration.

Folder types:

- `stylized_nature_megakit/`: imported third-party CC0 nature assets from
  Quaternius, converted into editable Blender sources and runtime glTF models.

For original Elderforge assets that we expect to redesign directly, prefer the
source-backed folders under `assets/models/props/`, `assets/models/resources/`,
or `assets/models/refining_stations/`. Third-party environment packs should keep
their license text and raw source input beside any converted runtime models.
