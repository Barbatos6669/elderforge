# Prop Model Sources

Reusable world props that may be redesigned later should live here with editable
Blender source and stable runtime exports.

- `ruin_arch/`: low-poly stone arch with runes, vines, source `.blend`, and
  runtime `.glb`.
- `barbatos_props/`: custom Elderforge props made by the project owner.
- `free_3d_modular_game_assets_for_prototyping/`: CC0 modular prototype pieces,
  editable Blend sources, and runtime glTF exports.
